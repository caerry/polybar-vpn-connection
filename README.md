This script allow you to display the current VPN status
It uses typical VPN interface and nmcli for getting active connections

The support list:

- Wireguard
- Openvpn 
- OpenConnect
- VPNC
- PPTP
- L2TP
- SSTP
- IPSec 
- Generic VPN
